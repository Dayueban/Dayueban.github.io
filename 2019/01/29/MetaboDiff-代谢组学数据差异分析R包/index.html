导读

  • 比较非靶向代谢组学是向质谱分析领域之外的研究人员通过商业供应商提供代谢组学成熟起来的。MetaboDiff包的目的是从代谢物的相对含量表开始，为差异代谢组学分析提供一个比较简单入门的的方法。此外，还介绍了加权相关网络分析在代谢组学中的应用。
  • R包：MetaboDiff 0.9.2

Getting started

  • MetaboDiff包需要R >=3.4

Installing dependencies

在安装MetaboDiff之前需要安装好WGCNA。

然后安装MetaboDiff，可以通过Github安装

Part I: Data processing

Input data

MetaboDiff的输入文件包括三个

  • assay - 是一个包括代谢物相对丰度的矩阵
  • rowData - 包含可用代谢物注释的数据框
  • colData - 一个包含样品信息的数据框

本教程的示例数据来源于Priolo及其同事的一项研究,作者使用了Metabolome®比较61份前列腺癌25正常前列腺标本组织代谢物

注意请确保assay的列名 (sample)对应于colData的行名 (patients)，assay的行名对应于rowData的行名

该包的函数create_mae合并三个对象为一个综合对象MultiAssayExperiment对象，从而简化下游的分析

Metabolite annotation

如果HMDB，KEGG或ChEBI id是rowData对象的一部分，则可以从小分子途径数据库（SMPDB）检索代谢物注释

Imputation of missing values

与其他高通量技术相比，缺失值在定量代谢组数据集中很常见。na_heatmap函数可视化了样本中缺失的代谢物测量值。但是需要指定用于分组的colData中的列的名称和这两组的标签颜色。

填充函数用到knn_impute

Outlier heatmap

在数据归一化之前，我们希望排除数据集中存在的异常值。为此，我们提供了outlier_heatmap函数。该函数提供了参数k的设置，用于识别缺失值集群

从图形可以看出cluster=1因为丰度值较低，是缺失值，那么后期可以通过函数remove_cluster将该簇的个体移除

Normalization

方差稳定归一化（vsn）的方法用于保证方差在测量的质谱图上保持几乎恒定

这一步归一化之后，对象MultiAssayExperiment包含4个数据集

  • raw - raw relative metabolic measurements as provided by company or core facility
  • imputed - imputed relative metabolic measurements (k-nearest neighbor imputation)
  • norm - normalized relative metabolic measurements (vsn)
  • norm_imputed - normalized and imputed relative metabolic measurements (vsn)

Quality control of normalization

查看四个数据集的质量值分布

Part II: Data analysis

下面这个表格总结了MetaboDiff包能回答的生物学问题

Unsupervised analysis

为了以无监督的方式探索样品之间的代谢组学特征，MetaboDiff提供PCA作为线性和tSNE作为非线性降维方法

Hypothesis testing

组间代谢物差异检验用的函数是diff_test，如果是两个组，则用student t’ test，如果大于两个组，则用方差检验。并且P值经过多重检验校正。

比较分析的结果可以用火山图进行展示

Metabolic correlation network analysis

Implementation

得到有意义的子代谢通路，并且这些子通路是在不同的组之间富集的。MetaboDiff可以生成代谢相关网络。在MetaboDiff中，代谢相关网络分析由一下4组函数分析完成

  • diss_matrix - construction of dissimilarity matrix
  • identify_modules - identification of metabolic correlation modules
  • name_modules - name metabolic correlation modules
  • calculate_MS - calculation of module signficance to relate sample traits to modules

Association of sample traits with correlation modules

以下图表说明了感兴趣的样本性状（肿瘤与正常）与从数据中得到的代谢相关性模块的关联。选项p_adjust允许选择校正或者未校正的P值。
