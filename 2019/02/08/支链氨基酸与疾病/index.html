支链氨基酸介绍

人体蛋白质主要由20种氨基酸合成，其中9种被认为是“必需的”，因为它们不能从人体内的其他代谢物合成。其中有三种是支链氨基酸（BCAAs）：亮氨酸（Leu），异亮氨酸（Ile）和缬氨酸（Val），因其侧链为支链而非直链脂肪族而得名。最富含BCAAs的食物来源有肉类，鱼类，乳制品和鸡蛋。近几十年来，BCAAs已被研究作为在运动训练期间，在恶病质（肌肉萎缩）和衰老的综合征中增强肌肉蛋白质合成和质量的试剂。在这种背景下，已知Leu可以激活合成代谢信号分子mTORC1（哺乳动物雷帕霉素复合物靶点1）以及蛋白质合成中涉及的其他因子。

与BCAAs在负能量平衡条件下的潜在健康促进作用相反，在患有肥胖相关病症如胰岛素抵抗，2型糖尿病（T2D）和心血管疾病（统称为称为心脏代谢疾病）的人血液中观察到BCAAs的逐渐增加。1969年，卡希尔等人报道，在年龄和性别配对的个体中，肥胖个体与正常个体相比，血液中三种BCAAs和芳香族氨基酸，苯丙氨酸（Phe）和酪氨酸（Tyr）的浓度增加，甘氨酸（Gly）减少。BCAAs的增加与循环中的胰岛素水平相关，表明BCAAs水平的上升是胰岛素抵抗的一种表现。随后，与肥胖相关的循环中的和组织种脂质浓度的升高（血脂异常）成为焦点，研究表明脂质可以作为胰岛素抵抗和T2D的驱动因素。这些研究在很大程度上掩盖了对BCAAs在疾病发病机制中的作用的研究。

代谢组学技术的兴起

更加全面地认识到BCAAs与慢性人类疾病之间的联系，恰好代谢组学作为研究人类疾病工具的出现。与胰岛素敏感的个体相比，基于质谱的代谢组学的应用恰好重复了循环水平的BCAAs的变化情况，如Tyr和Phe的在肥胖且胰岛素抗性个体中丰度增加，而Gly减少。此外，代谢组学数据的主成分分析显示，与几种脂质相关的代谢物簇相比，BCAA相关代谢物簇与胰岛素抵抗的关系更为明显-这一发现已在多项人群研究中得到证实。BCAAs的增加，Phe和Tyr的循环水平也与高达5倍的T2D发展的风险增加相关，并且可以预测胰岛素敏感性在肥胖个体的饮食和行为减肥干预后得到改善。此外，尽管体重减轻，但接受胃旁路手术的肥胖个体的循环BCAAs下降幅度大于进行饮食干预的个体。这与响应于外科手术干预的葡萄糖稳态的更好改善相关。最近这些BCAAs与心脏代谢疾病表型相关的例子提出了一个重要的问题，即循环BCAAs是否在这些疾病中充当生物标志物，致病因子或两者都是。

为什么循环水平的BCAAs会增加

循环水平的BCAAs的丰度增加部分原因是脂肪组织中BCAAs的氧化速率降低了，BCAAs分解酶的协同转录抑制，以及肝脏中支链酮酸脱氢酶复合体（branched-chain ketoacid dehydrogenase,BCKDH）的磷酸化和活性丧失增加，所以导致从血液中吸收的BCAAs的量就少了。肥胖的啮齿动物模型表现出抑制性BCKDH激酶——BDK的在肝表达增加，并且活化的BCKDH磷酸酶PPM1K的表达降低。尽管哺乳动物组织缺乏从头BCAA合成所需的酶，但肠道内的细菌可以进行这些反应，并且与正常个体相比，肥胖个体的肠道微生物中能诱导整个BCAA生物合成途径。此外，将来自肥胖个体的肠道微生物移植到无菌小鼠（其缺乏肠道微生物）中导致循环BCAA的显著增加。但是需要进一步的研究来确定细菌来源的BCAAs对这些代谢物的循环池的贡献。

最近的研究表明，与血脂异常和胰岛素抵抗相关的人类遗传变异与循环BCAAs增加密切相关。然而，特异性t提高循环BCAAs的遗传变异，包括PPM1K基因附近的突变，与胰岛素抵抗无关。尽管如此，在大型队列分析中，Ile，Leu或Val循环中增加1个标准差就与发生T2D的风险明显增加相关。模型研究表明肥胖和胰岛素抵抗是肥胖相关的循环BCAAs升高的主要驱动因素，与1969年Cahill等人的结论一致。但也认为当BCAAs在肥胖环境慢性升高时，其可能有助于T2D发病机制。

在啮齿动物模型中已经进行了许多关于膳食补充或限制BCAAs的效果的研究，结果并不一致。不一致的部分原因在于该研究是否涉及对所有三种BCAA的调控或仅涉及Leu。单独补充Leu会导致其他两种BCAAs的浓度在循环中减少，因此不能模拟人类心脏代谢疾病状态，即所有三种BCAA都升高。

减少膳食来源的BCAAs水平可以提高肥胖啮齿动物模型中的胰岛素敏感性和葡萄糖稳态，与肌肉中多种脂质来源的代谢物的正常化和呼吸交换率降低（RER;产生的CO2与消耗的O2的比率）相一致，表明脂肪酸氧化效率提高（fatty acid oxidation, FAO）。相反，与单独喂食HF饮食的大鼠相比，喂食补充有BCAAs（HF / BCAA）的高脂肪饮食的大鼠减少了食物摄入和体重，但是促进了胰岛素抵抗和骨骼肌中脂质的积累。用HF饮食配对喂养以匹配较低的HF / BCAA消耗率，或喂食补充有BCAAs的低脂饮食的大鼠不产生胰岛素抗性。与瘦大鼠相比，肥胖大鼠的肌肉Gly浓度降低，但在喂食BCAA限制饮食的肥胖大鼠中Gly的水平变得正常了，证明了BCAA和Gly代谢的直接相互作用，这可以解释它们在研究中相关性是相反的原因。肌肉Gly水平与尿液中乙酰甘氨酸水平相关，这意味着酰基甘氨酸加合物的形成可以促进从肌肉中去除过量的脂质来源的代谢物以改善胰岛素敏感性。与减轻BCAA分解代谢中肥胖相关损伤可以促进提高FAO和代谢健康功效的想法相一致，将BDK，BT2小分子抑制剂或表达PPM1K的重组腺病毒（AdCMV-PPM1K）给予Zucker-肥胖大鼠可以降低循环BCAAs，改善葡萄糖耐受和胰岛素敏感性，并降低肝脏中RER和脂肪的过量储存。

为什么调控BDK/PPM1K比例具有如此广泛的代谢作用？部分在于新发现的这种激酶-磷酸酶对的替代底物，包括三磷酸腺苷（ATP）-柠檬酸裂解酶（ACL），一种从头脂肪生成（DNL）的关键酶，与乙酰辅酶A（CoA）羧化酶（ACC）和脂肪酸合成酶（FAS）共同发挥作用。尽管BDK会抑制BCKDH的磷酸化，但磷酸化作用可以激活ACL。ACL产生细胞溶质乙酰辅酶A，形成丙二酰辅酶A，作为DNL的直接底物和FAO的变构抑制剂。因此，预计BT2治疗或PPM1K过表达会使ACL失活，导致DNL降低，FAO增加，肝脏脂肪积累减少，这与肥胖大鼠的发现一致。

长期以来，含糖食品和饮料的消费被认为是代谢疾病的风险因素，果糖通过其通过碳水化合物反应元件结合蛋白（ChREBP）转录因子诱导DNL的能力而具有特别突出的作用。喂食高果糖饮食或肝脏过度表达ChREBP导致正常体重大鼠肝脏中BDK的表达增加和PPM1K表达的降低。另外，BDK和ChREBP转录水平与来在患有非酒精性脂肪肝病（NAFLD）患者的肝脏样品中紧密相关。这些观察结果表明，营养过剩，特别是涉及高果糖饮食，激活ChREBP以增加BDK / PPM1K比例，从而抑制BCKDH活性和增加循环BCAAs同时促进DNL和肝脏脂肪积累。

BCAAs导致代谢疾病的机制初探

BCAAs可能导致心脏代谢疾病表型的直接机制是什么？在心肌细胞中，葡萄糖通过抑制环状腺苷一磷酸应答元件结合蛋白（CREB）介导的Kruppel样因子15（KLF15）转录因子（编码BCAA分解代谢酶的基因的全局激活因子）来抑制BCAA分解代谢。由此产生的BCAAs积累可激活mTORC1以驱动蛋白质合成和不健康的心脏肥大，这与抑制BDK可以改善，而Ppm1k遗传缺失则会损害心力衰竭小鼠模型中的心脏功能的研究相一致。在肝脏中，BCAA限制激活eIF2α激酶GCN2以参与氨基酸保守程序，同时通过抑制另一种DNL诱导转录因子，类固醇反应元件结合蛋白-1c（SREBP-1c）抑制DNL基因表达。此外，对过氧化物酶体增殖物激活受体γ共激活因子1α（PGC-1α）的强制表达的响应可以使骨骼肌中产生Val衍生的代谢物3-羟基异丁酸（3-HIB），导致调控反式内皮脂肪的运输。循环3-HIB浓度与糖尿病患者的血糖正相关，这表明该代谢物可促进骨骼肌中过多的脂质储存和胰岛素作用受损。然而，在使用BT2激活BCKDH的研究中，预期会引起循环3-HIB-胰岛素抵抗的治疗在Zucker肥胖大鼠中得到改善而不是加剧，而肌肉组织中的脂质衍生代谢物没有变化。这并不排除3-HIB在治疗干预前对大鼠的脂质积累和胰岛素抵抗的发展的可能性。有必要进一步研究这种机制。

结语

因此，BCAAs显然是心脏代谢疾病表型的生物标志物。也有越来越多的证据表明，它们参与疾病的发病机制，主要是在肥胖的背景下。膳食来源的糖，可以刺激碳水化合物反应元件结合蛋白（ChREBP），用于增加BDK/PPM1K的比例，与此同时，抑制了BCAAs的分解代谢，激活了DNL（重头脂肪生成）作用和抑制了FAO（脂肪酸氧化）作用。因为BDK和PPM1K调节ACL，BCAAs作为血脂异常的生物标志物，其本身有助于心脏代谢疾病。 通过分子或者药理学的方法降低BDK / PPM1K比例足以逆转肥胖啮齿动物的疾病表型。此外，在致肥胖但非低脂肪饮食中补充BCAAs，或者它们对喂食肥胖啮齿动物的饮食的限制，对代谢健康有影响。最后，将遗传变异使得BCAA水平升高与T2D风险增加的相关性与BCAAs在T2D发病机理中的因果作用一致。正在进行的关于BCAAs和相关代谢物影响心脏代谢疾病发病机制的机制研究是必要的，并将有助于这一新兴领域的发展。
